# AK-25 · E8 root system as multi-scale vortex structure

## Approach tried

Model multi-scale strain-vorticity configuration as root system
of exceptional Lie algebra E8.

## Obstruction

NS multi-scale structure is **continuous and dynamical**
(parabolic evolution).  Root systems are **discrete and
algebraic** (finitary).  Liouville theorem exploits anti-
coherence (scales decoupling).  Root systems are maximally
coherent.  Opposite structural assumptions.  No algebraic
embedding of NS dynamics into E8 exists.

exhaustion).
